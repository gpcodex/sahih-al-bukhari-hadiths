== Description ==

- Sahih al-Bukhari is a collection of hadith compiled by Abu Abdullah Muhammad Ibn Isma`il al-Bukhari(rahimahullah). 
- His collection is recognized by the overwhelming majority of the Muslim world to be one of the most authentic collections of the Sunnah of the Prophet Salla Allah `Alaihi Wa Sallam. 
- It contains roughly 7563 hadith (with repetitions) in 97 books.
- The translation provided here is by Dr. M. Muhsin Khan.

You just have to write the shortcode [bukhari-hadiths] on your new page.
It is better to put your page in fullwidth

== Installation ==


1. (Recommanded) install the plugin from the administration.
plugin-> add new-> in "Search Plugins" tape: Sahih al-Bukhari Hadiths
(if you manually install consider changing chmod on the ftp, otherwise you risk having errors.)

2. Activate the plugin through the 'Plugins' menu in WordPress

3. use shortcode [bukhari-hadiths] on your new page

4. it's recommended to have a full width page.
